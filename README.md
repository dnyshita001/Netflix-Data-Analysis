# Netflix-Data-Analysis

## Project Overview

This project analyzes the Netflix Titles dataset using Python and Pandas.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Cleaning

* Handled missing values
* Converted date_added to datetime format
* Removed invalid records
* Checked for duplicates

## Exploratory Data Analysis

Questions answered:

1. Movies vs TV Shows
2. Top countries producing content
3. Most common ratings
4. Release year trends
5. Most popular genres

## Visualizations

* Bar Chart
* Line Chart
* Histogram
* Pie Chart
* Scatter Plot
* Heatmap

## Key Insights

* Movies dominate Netflix's catalog.
* The United States contributes the most content.
* TV-MA is the most common rating.
* Content production increased significantly between 2016 and 2020.
* International dramas are among the most popular genres.
